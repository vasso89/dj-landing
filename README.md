# DJ Nova Pulse – Music Landing Page

## Description
A responsive landing page for a DJ / electronic music brand.  
Built as a practice project using a reference design and custom layout decisions.

## Features
- Responsive design (Desktop, Tablet, Mobile)
- Flexbox & CSS Grid for layout
- Custom burger menu (CSS only)
- Modern dark UI style
- Semantic HTML5 structure

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Normalize.css
- Google Fonts (Poppins, Work Sans)
